# Water Exploitation Dashboard (WEI+)

This project presents an interactive data dashboard developed using Streamlit to analyse water stress levels across countries using the Water Exploitation Index Plus (WEI+) dataset.


## Dataset
- **Name:** Water Exploitation Index Plus (WEI+)
- **Source:** European Environment Agency (EEA)
- The dataset includes:
  - Country
  - Year
  - Quarter
  - Water Consumption
  - Water Abstraction
  - WEI (%)


## Project Objectives
- Analyse water stress trends across countries over time  
- Identify seasonal patterns in water consumption  
- Compare country performance with global averages  
- Provide interactive visualisations for decision-making  


## Features
- Interactive filters (Country, Year, Quarter)
- Key Performance Indicators (KPIs)
- WEI trend line chart
- Quarterly consumption bar chart
- Top 10 countries by WEI
- Global water stress map
- Dynamic data table


## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- NumPy
- Matplotlib

## How to Run the App Locally

Navigate to the folder:
cd WEIProject

Install dependencies:
pip install -r requirements.txt

Run the app:
streamlit run app.py
